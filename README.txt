soom 숨 — 친환경 힐링 라이프 쇼룸 홈페이지
=================================================

[ 실행 방법 ]
- index.html 파일을 더블클릭하면 브라우저에서 바로 열립니다. (별도 설치/빌드 불필요)

[ 폴더 구조 ]
- index.html                  메인 페이지
- assets/css/styles.css       스타일(디자인 토큰 · 반응형 포함)
- assets/js/main.js           동작(메뉴, 스크롤 애니메이션, 필터, 영상 모달, 폼)
- assets/                     영상/이미지 파일을 넣는 곳
- assets/img/                 제품·포스터 이미지 폴더

[ 메인 영상 넣기 ]
1. 원하는 mp4 영상을 준비합니다. (가로형 권장, 10~20MB 이하면 로딩이 빠릅니다)
2. 파일명을 hero.mp4 로 바꿔서 assets/ 폴더에 넣습니다.
   → assets/hero.mp4
3. 페이지를 새로고침하면 히어로 영상이 자동 재생됩니다.
   ※ 영상 파일이 없으면 자연 그라데이션 배경이 대신 보이도록 만들어 두었습니다.

[ 영상 첫 화면(포스터) 이미지 ]
- assets/img/hero-poster.jpg 로 이미지를 넣으면 영상 로딩 전/모달에 표시됩니다. (선택)

[ 제품 사진으로 교체하기 ]
- 지금은 제품 카드 이미지가 색상 그라데이션으로 표시됩니다.
- 실제 사진을 쓰려면 styles.css 의 `.product-card__media[data-tone="..."]`
  규칙에서 background 를 아래처럼 이미지로 바꾸면 됩니다.
    background: url("../img/제품파일명.jpg") center/cover no-repeat;
- 또는 index.html 의 각 카드 <div class="product-card__media" ...> 안에
  <img src="assets/img/제품.jpg" alt="제품 설명"> 를 넣어도 됩니다.

[ 내용 수정 ]
- 브랜드명/문구/제품/가격/주소 등은 index.html 에서 직접 텍스트를 바꾸면 됩니다.
- 색상 톤은 styles.css 상단의 :root 변수(--forest, --sage, --clay 등)만 바꾸면
  사이트 전체 색이 한 번에 바뀝니다.

[ 포함된 섹션 ]
1. 상단 공지바
2. 네비게이션 (모바일 햄버거 메뉴 포함)
3. 히어로 — 메인 영상 + 카피 + CTA
4. 브랜드 스토리 (고객이 보는 정보) + 가치 카드
5. 통계 지표 (스크롤 시 숫자 카운트업)
6. 제품 카드 (카테고리 필터: 전체/리빙/아로마/케어)
7. 쇼룸 안내 + 방문 예약
8. 저널/뉴스레터 구독
9. 푸터 (연락처/소셜/약관)

접근성(대비·키보드·포커스·스크린리더), 반응형(모바일~데스크톱),
prefers-reduced-motion(모션 최소화 설정) 을 모두 고려해 제작했습니다.
